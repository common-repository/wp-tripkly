== Description ==

[tripkly id="854"] shortcode

Show Tripkly Trails activity on your WordPress site.
You can use easy tripkly shortcode for embed trail and map from tripkly.com

= Useful: =
* Use shortcut
Insert the shortcode [tripkly id="854"] in your posts or pages.

== Other Notes ==

= Params: =
* **id** - id is track number from tripkly.com site. You can found in url (example: http://www.tripkly.com/community/trails/850/vajont/ -> id=850 )
